# Actividad - Ejercicios de práctica 1 - IU DIGITAL DE ANTIOQUIA

## Objetivo del proyecto
El objetivo de esta actividad es aplicar los conocimientos de la Unidad 1 mediante el uso de arreglos unidimensionales en Java. El programa solicita al usuario ingresar datos para llenar dos arreglos de 7 posiciones, genera un tercer arreglo calculado a partir de la diferencia de los dos primeros, muestra sus datos en pantalla y calcula el promedio resultante.

## Instrucciones de ejecución
Para ejecutar este proyecto en tu entorno local, sigue estos pasos:

1. Asegúrate de tener instalado **Java Development Kit - JDK** en tu computadora.
2. Clona este repositorio o descarga el archivo fuente `.java`.
3. Abre una terminal o línea de comandos y navega hasta la carpeta donde guardaste el archivo.
4. Compila el archivo ejecutando el siguiente comando:
   `javac src.Main.java` *(Cambia "src.Main.java" si nombraste tu archivo de otra manera)*
5. Ejecuta el programa con el comando:
   `java src.Main`
6. Sigue las instrucciones que apareceran en la consola.
